# South Shore Landscape Directory

This package is structured for GitHub Pages and static hosting.

## Upload to GitHub

Upload the contents of this folder directly to the root of your repository, not inside another folder. The repository should show `index.html`, `styles.css`, `script.js`, and the `assets/` folder at the top level.

Correct:

```
index.html
styles.css
script.js
assets/
  south-shore-landscape-directory-logo.png
```

Incorrect:

```
south-shore-landscape-directory/
  index.html
```

## GitHub Pages

In GitHub, go to Settings → Pages, choose Deploy from branch, select `main`, and save.
